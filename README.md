Мобильное приложение для бронирования авиабилетов
Требования к системе 
Минимальные требования  
ОС: Android 7.0 (Nougat) или выше (API Level 24+) 
RAM: 1 ГБ оперативной памяти  
Память: 50 МБ свободного места  
Интернет: Требуется на постоянной основе
Рекомендуемые требования 
ОС: Android 10 или выше (API Level 29+) 
RAM: 2 ГБ оперативной памяти 
Память: 100 МБ свободного места 
Интернет: Стабильное соединение Wi-Fi или мобильная сеть 

Архитектура:
Клиент: Android
Язык: Kotlin
База данных: Room Database, Sqlite(внутреняя) / Интеграция с backend
Требования: Android SDK 24+, Android Studio Arctic Fox и выше
MVVM Model-View-ViewModel - архитектурный паттерн

Установка:
Клонировать репозиторий
Открыть проект в Android Studio
Дождаться синхронизации Gradle
Запустить на эмуляторе или физ.устройстве
